Passos para criar um projeto corretamente

  • Criar a pasta onde vai esta o projeto.
  • Abrir a pasta no terminal
  • Digitar o comando: git init (para inicializar o repositorio local do git)
  • Trabalhar no projeto normalmente
  • Fazer o commit do que foi feito
  • Depois so precisa fazer uma unica vez esse processo:
  • Criar repositorio no Github
  • Linkar o repositorio local com o repositorio do Github
